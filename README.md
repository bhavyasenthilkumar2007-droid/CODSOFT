# CODSOFT
tic-tac-toe AI


This project is a simple Tic Tac Toe game developed using Python. Two players can play the game by selecting positions from 1 to 9.

Features

- Console-based game
- Two-player support
- Position validation
- Simple board display

Technology Used

- Python
